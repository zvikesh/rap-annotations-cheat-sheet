# Header Level Annotations

```
-- DCL
@AccessControl.authorizationCheck: #CHECK
-- Metadata
@Metadata:{
 allowExtensions: true,
 ignorePropagatedAnnotations: true
}
-- Data Model
@VDM.viewType: #BASIC
-- Performance
@ObjectModel.usageType:{
    serviceQuality: #A,
    dataClass: #CUSTOMIZING | #ORGANIZATIONAL | #MASTER | #TRANSACTIONAL,
    sizeCategory: #S (#CUSTOMIZING) | #M (#ORGANIZATIONAL) | #L (#MASTER) | #XL  (#TRANSACTIONAL)
}
--- Analytical
@Analytics : {
    dataCategory: #DIMENSION, 
    dataExtraction.enabled : true | false ?Reference?
}
```

# Field Level Annotations

**Labels**

```
@EndUserText:{ label: 'Airline ID', quickInfo: 'Airline ID' }
CarrierID
```

**Semantic Annotations**

Semantic Annotations are for Consumption View reusability

```
-- Currency
@Semantics.amount.currencyCode:'currency_code' 
price : abap.curr;
@Semantics.currencyCode: true                    //Not allowed in View Entity
currency_code : abap.cuky;

-- Quantity
@Semantics.quantity.unitOfMeasure:'' 
distance : abap.int4;
@Semantics.unitOfMeasure: true                   //Not allowed in View Entity
distance_unit : msehi;

-- Administration
@Semantics.user.createdBy: true
created_by : syuname;
@Semantics.systemDateTime.createdAt: true
created_at : timestampl;
@Semantics.user.lastChangedBy: true
last_changed_by : syuname;
@Semantics.systemDateTime.lastChangedAt: true    //Used for Locking
last_changed_at : timestampl;

-- 
@Semantics.businessDate 
@Semantics.fiscal Information on the fiscal year

```

**? --- Foreign Key or Text View**

```
@ObjectModel.association.type
@ObjectModel.text.Association
@ObjectModel.foreignKey. association
-- Text View
@Semantics.language //Language Key
@Semantics.text     //Text Value
```
